これはデザインパターンを勉強するためにデザインパターンを使って作ってみたウェブページと画像をダウンロードするだけのJavaのクローラーです。
深さを指定することでページのリンクを辿ってリカーシブにページを取ってくる事が可能です。
シングルスレッドで動作するのでとても遅いですがこれからスレッドを使うように作り替えていく予定です。
classpathにJerichoHTMLとLog4Jを追加して
java Stater http://hostname.com/example/ 2
といったようにして実行してください。
この2というのは深さの指定です。
